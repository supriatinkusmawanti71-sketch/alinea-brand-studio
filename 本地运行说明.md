# Alinea 2.0 本地运行说明

这个文件夹是 Alinea Brand Agent Studio 的本地源码包。

## 包内内容

- `apps/web`：前端页面。
- `apps/api`：后端 API 和任务入口。
- `backend`：三 Agent 工作流、模型供应商、导出逻辑。
- `infra`：网关等基础设施配置。
- `compose.yaml`：本地 Docker 编排。
- `.env.example`：环境变量模板。
- `package.json` / `package-lock.json`：前端依赖配置。
- `pyproject.toml` / `uv.lock`：Python 后端依赖配置。

## 不包含的内容

为了安全和体积，打包时已排除：

- `.env`：里面可能有 API Key。
- `.git`：开发历史。
- `node_modules`：前端依赖目录。
- `.venv`：Python 虚拟环境。
- 缓存、临时文件、历史生成资产。

## 启动方式

1. 安装 Docker Desktop。
2. 在项目根目录复制环境变量模板：

```bash
cp .env.example .env
```

3. 打开 `.env`，填入真实模型供应商配置和 API Key。
4. 启动服务：

```bash
docker compose up --build
```

5. 打开浏览器访问：

```text
http://127.0.0.1:8080
```
